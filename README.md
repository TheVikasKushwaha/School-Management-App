# School Portal System

A complete School Management Portal that provides features for students, teachers, and administrators to manage academic activities efficiently.

## 🚀 Features
- 🔐 User Authentication (Login & Signup)
- 👨‍🏫 Teacher Management
- 📢 Notice Board System
- 📅 Event Management
- 📞 Contact/Enquiry Form
- 🛠️ Admin Panel Dashboard
- 📊 CRUD Operations
